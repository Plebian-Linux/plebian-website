Plebian — PINE Live Debian

About

Plebian is a set of Debian-based live images for PINE64's Quartz64 and SOQuartz line of devices. Plebian aims to be unsurprising, well documented and easy to get started with.

Learn more

Installation

Since Plebian images are "live" images, there is no installer. Installing Plebian onto your Quartz64 device is as simple as flashing the right image to either your SD or eMMC storage.

Get started

Contributing

Plebian's image generation pipeline is completely open, which means you can not only make your own changes to it, but also contribute them back.

Found an issue you want to report or have a suggestion to make? Head on over to our GitHub repository!
